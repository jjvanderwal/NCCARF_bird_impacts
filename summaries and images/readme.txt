Information on use of NCCARF Bird NARP project summary scripts.

00.scripts
----------
These generate base data upon which following script rely.

All following scripts must be run in order of number.  Where a batch script and a run script share a number, the batch script must be run.

batch scripts
---------------
Batch scripts submit multiple jobs in batch (non-interactive).

run scripts
-----------
These scripts are submitted and run by the batch scripts.  These must be saved to a directory on the HPC.
